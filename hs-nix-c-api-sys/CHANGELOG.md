## [0.1.0.0]

Auto-generated Haskell FFI bindings to the complete Nix C API surface,
generated by `hs-bindgen`.

- Bindings to all six Nix C libraries: `nix-util-c`, `nix-store-c`, `nix-expr-c`, `nix-fetchers-c`, `nix-flake-c`, `nix-main-c`.
- Each library provides type definitions, safe FFI, unsafe FFI, and function pointer wrapper modules.
